Deskripsi 

Ini adalah program yang sudah Devan kerjakan, jumlahnya ada 8.
Semuanya berkaitan dengan Matematika dan program tersebut Devan kerjakan selama 3 tahun, selama Devan SMP.

Devan memakai python dalam mengerjakan semua program tersebut.
Selanjutnya, Devan sedang mencoba untuk membuat program HTML, agar lebih mudah dan lengkap.
Begitu juga program Geometri, Devan mencoba tanpa memakai streamlit.

Selain itu, ada 2 program yang saat ini Devan ingin kembangkan yaitu:
- Program Equation Solver versi 5
- Program Geometri versi 1.6

Dan juga sedang mengembangkan halaman timeline dan updates.

Di halaman Hasil Karya sudah menambahkan Canva, Google Form, Kumpulan Website, dan Hasil Karya Kelas 7, sehingga versinya berubah menjadi v1.7.3.

Font untuk heading sudah berubah dari Raleway ke Inter, dan menunya sudah diperbaiki dan diperlengkap, sehingga versinya berubah menjadi v1.7.4.

Di versi 1.9, saya menggunakan Google Search Console supaya bisa mencari website saya dengan lebih mudah. Selain itu, di versi 1.9.1, ada fitur tema gelap dan pencarian (search) Google yang baru. Di versi selanjutnya, saya memeratakan tampilan website yang baru dan melengkapinya dengan gambar transparan supaya lebih menarik, serta memperlengkap halaman blog.
